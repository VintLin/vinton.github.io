阅读

大部分业余时间留给了阅读，目前比较喜欢社科类、心理学、方法论相关书籍，偶尔会阅读一些文学作品。这些是对我影响比较大的作家艾里希•弗洛姆、纳西姆•塔勒布、约瑟夫•坎贝尔、理查德•道金斯、罗曼·罗兰。目前为止我最喜爱的几本书：

  • 《自私的基因》理查德•道金斯 # 我们不过是基因的载体，它只关心自己是否可以传承下去，而不关心作为载体的利益。
  • 《世界观》理查德•德威特 # 世界观是一组相互联结的，像拼图一样的观点体系。
  • 《禅与摩托车维修艺术》罗伯特•波西格 # 嬉皮士通过越野摩托车之旅探讨技术、哲学和精神的交集。
  • 《有限与无限的游戏》詹姆斯•卡斯 # 有限游戏以取胜为目的，而无限游戏以延续游戏为目的。
  • 《人间游戏》Eric Berne # 所有人类的互动都可以被理解为一场具有一定规则和策略的“游戏”。
  • 《逃避自由》艾里希•弗洛姆 # 现代社会创造的“自由”实际上只是一种幻觉。
  • 《思考，快与慢》丹尼尔•卡尼曼 # 大脑有两种截然不同的思维系统:一种是快速的、直觉的，另一种是缓慢的、分析的。
  • 《心流》米哈里•契克森米哈赖 # 心流是完全专注做事而全然忘却时间的状态，这种状态是幸福的关键。
  • 《荒原狼》赫尔曼•黑塞 # 人和社会环境永远存在矛盾。
